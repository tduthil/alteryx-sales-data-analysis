
Analyzing Sales Data with Alteryx
Sales Analysis | Data Preparation & Workflow Automation

📌 Project Overview
This project simulates a real-world business scenario in which a data analyst is tasked to prepare, transform, and analyze sales data using Alteryx; answering a series of strategic business questions through a structured, documented workflow.
The project demonstrates end-to-end data pipeline construction: from raw file ingestion and cleaning, through joins and transformations, to aggregated outputs ready for business consumption.

🎯 Business Objective
DC Industries needed actionable insights from their sales data but faced common data challenges: siloed files, inconsistent formatting, and no unified view across regions and order details. The goal was to consolidate this data and answer key performance questions through a repeatable, well-documented Alteryx workflow.

📋 Workflow Best Practices Applied

✅ Documented workflow with annotations throughout for clarity and maintainability

✅ Segmented workflow into containers to allow modular execution and future flexibility 

✅ Structured outputs designed for direct handoff to business stakeholders

Combining & Enriching Customer Order Data

Unioned two separate customer order files into a single consolidated dataset
Appended regional data to enable geographic performance analysis
Delivered an additional output file with order counts broken down by region following a follow-up request from leadership

Transforming Order Details

Shifted focus to the order details file, applying data cleansing and transformation steps to standardize the dataset
Joined the cleaned order details to the consolidated customer data
Produced master data files to serve as the foundation for downstream analysis

Answering the Business Questions

Aggregated and transformed the master data four separate ways, each cut targeting a specific business question
Delivered a complete findings package to DC Industries leadership with insights ready for decision-making
